Minhas Notas

Aqui você pode deixar trechos de código ou anotações sobre esta página e depois acessá-los em qualquer lugar. Basta inserir seu nome e uma senha!

Recomendações

Escolha uma boa senha para que alunos com o mesmo nome que o seu não acessem por engano as suas anotações. Caso você escreva seu nome ou a senha errados, uma nova conta é criada, então cuidado na hora de digitar para não "perder" suas notas!

Exemplos de senhas: Numero do celular, data de nascimento, nome do cachorro, fruta preferida, etc.

Atenção: Nós mantemos um registro dos dados de login para possibilitar a recuperação de notas "perdidas". Favor levar isso em consideração na hora de digitar sua senha!

Nunca use a senha do seu e-mail!

Evite passar sua senha para outros alunos. Caso queira compartilhar novas anotações com alguém, escolha um novo nome e/ou senha.

As notas são exibidas de acordo com a página atual. Para acessar uma lista das suas anotações clique no botão , clique em para baixar todas elas. Use para recuperar sua senha.

Este recurso é baseado no serviço http://dontpad.com/ e Google Forms, não abuse!
